**Capybara Meow: Học Toán Vui Nhộn** là một tựa game endless-runner giáo dục mã nguồn mở chạy trực tiếp trên trình duyệt. Trò chơi kết hợp sự dễ thương của nhân vật Capybara và chú Mèo cam (Meow) với các thử thách toán học, giúp trẻ em từ 6-12 tuổi rèn luyện kỹ năng tính toán một cách đầy hứng thú.

🎮 **[PLAY THE GAME LIVE HERE] (https://chinhnt2k3.github.io/capybara-meow-game/)**

---

## ✨ Features (Tính năng nổi bật)

* 🧠 **Học Toán Tương Tác (Math Integration):** Thay vì chỉ né chướng ngại vật, người chơi phải tính toán nhanh để chọn đúng làn đường có đáp án chính xác.
* 🌍 **Hỗ Trợ Song Ngữ (Bilingual):** Hỗ trợ chuyển đổi mượt mà giữa **Tiếng Việt** và **English** ngay trong lúc chơi.
* 🏃 **Đồ Họa & Animation Tùy Chỉnh:** Sử dụng 100% HTML5 Canvas API để vẽ nhân vật, môi trường cuộn (parallax scrolling) và các hiệu ứng chuyển động nhấp nhô vui nhộn.
* 📈 **Độ Khó Tăng Dần:** Game tự động chuyển từ phép cộng cơ bản sang các phép toán có hai chữ số và phép trừ khi người chơi tăng cấp.
* 📱 **Nhẹ & Nhanh:** Không cần cài đặt, chơi ngay trên mọi trình duyệt web.

## 🕹️ How to Play (Cách chơi)

1.  **Mục tiêu:** Giúp Capy và Meow chạy càng xa càng tốt bằng cách vượt qua các Cổng Toán Học (Math Gates).
2.  **Thử thách:** Khi đến gần một Cổng Toán Học, hãy đọc kỹ phép tính trên bảng.
3.  **Điều khiển:**
    * Nhấn phím **Mũi tên Lên (⬆️)** để chuyển sang làn đường phía trên.
    * Nhấn phím **Mũi tên Xuống (⬇️)** để chuyển sang làn đường phía dưới.
    * *(Hãy chọn làn đường có chứa đáp án đúng!)*
4.  **Luật chơi:** * Chọn đúng: Cộng điểm, tăng tốc.
    * Chọn sai: Mất 1 trái tim ❤️. Hết 3 tim sẽ Game Over!

## 🚀 How to Run Locally (Chạy trên máy cá nhân)

Vì đây là một game thuần HTML/JS/CSS, bạn không cần cài đặt bất kỳ server hay thư viện nào.

1.  Clone repository này về máy:
    ```bash
    git clone [https://github.com/](https://github.com/)[your-username]/[your-repo-name].git
    ```
2.  Mở thư mục vừa tải về.
3.  Click đúp vào file `index.html` để mở game trên trình duyệt web của bạn (Chrome, Firefox, Safari...).

## 🛠️ Built With (Công nghệ sử dụng)

* **HTML5 Canvas:** Dựng khung trò chơi và render đồ họa.
* **Vanilla JavaScript (ES6):** Xử lý logic game, vòng lặp (game loop), hoạt ảnh và hệ thống sinh phép toán ngẫu nhiên.
* **CSS3:** Thiết kế UI (giao diện điểm số, nút bấm) và bố cục tổng thể.
